# Examples

Each subdirectory is a self-contained example with its own README, description file,
pre-generated schema, and extraction scripts.

| Example | Document type | What it extracts |
|---------|--------------|-----------------|
| [fidelity-brokerage-statement](./fidelity-brokerage-statement/) | Fidelity monthly brokerage statement | All transaction records from the Activity section (trades, dividends, cash, card, fees) |
